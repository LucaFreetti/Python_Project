# Travel Destinations Manager — Python Data Structures Project

## Objective
A Python project that simulates a travel agency management system.
The goal is to practice core Python concepts — dictionaries, lists, functions,
loops and lambda expressions — by building a small but functional tool
to store, filter and analyze a catalog of travel destinations.

## What It Does
The notebook builds a structured catalog of international destinations
and implements a set of functions to interact with it:

- **Add destinations** — register new cities or countries dynamically
- **List all destinations** — display countries and their cities in a clean format
- **Sort by price** — rank all destinations from cheapest to most expensive
- **Count destinations** — total count and breakdown by country
- **Calculate average price** — across the entire catalog
- **Find the cheapest destination** — using lambda and min()
- **Advanced filter** — filter destinations simultaneously by max days,
  max price, recommended season and accommodation type

## Concepts Practiced
- Nested dictionaries and lists
- Dictionary methods: `.keys()`, `.values()`, `.items()`
- `for` loops with nested iteration
- `isinstance()` for type checking
- Functions with optional parameters (`None` defaults)
- Lambda functions with `sorted()` and `min()`
- String formatting with f-strings

## How to Run
1. Clone the repository
2. Open the notebook: `jupyter notebook Luca_Frittitta_Python.ipynb`
3. Run all cells in order

No external libraries required — pure Python only.
